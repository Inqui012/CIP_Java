* 자바에서 실제 실행시키려는 코드(매서드)들은 파일안의 Class 안에 들어가있어야 한다.
    모든 자바 프로그램의 기본구조는 아래를 충족시켜야 한다.
    Main.java (파일) -> Public class Main { 매서드 }
    ex)
    public class Main {
        public static void main(String[] args) {
            System.out.println("Hello World");
        }
    }
    파일명 Main과 클래스명 Main은 동일해야 한다. (대소문자 구분, 반드시 대문자로 시작)

* 변수 지정
    (변수 타입) (변수이름) = (값);
    ex) String name = "John";
    이렇게 지정된 변수는 코드 안에서 얼마든지 다시 변경할 수 있다.
    변수 이름은 유지보수를 위해 알아보기 쉬운 이름으로 하는것이 좋다.
    변수가 담고있는 값을 설명하는 느낌의 이름이 가장 추천된다.
    소문자 영어로 시작하고, 스페이스 불가능, $, _ 사용가능, 대소문자 구분, 자바상에서 사용되는 int 나 float 같은 단어와 중복 불가능.
    값은 문자열일 경우 ""로, 문자일경우 ''로, 숫자일경우 그대로(예외있음), true false 도 그대로

    final (변수타입) (변수이름) = (값);
    이렇게 지정된 변수는 직접 코드를 바꿔쓰지 않는이상은 변경되지 않는다.

    변수를 여러개 지정할 경우 여러줄을 입력하지 않아도 축약하여 쓸 수 있다.
    ex) int x = 5, y = 6, z = 50; / 세 종류의 변수를 한줄로 지정
    ex) int x, y, z;
        x = y = z = 50; / 변수의 이름을 먼저 지정하고 추후에 값을 지정.

* 변수의 데이터 타입
    데이터타입이 저장가능한 범위는 byte수가 클 수록 넓어진다. 당연한가.
    Primitive [항상 수치를 가지고 있어야한다, 소문자시작, byte에 따라 크기가 다름]
        byte    / 1byte / 정수. 
        short   / 2byte / 정수. 
        int     / 4byte / 정수. 
        long    / 8byte / 정수. 저장시 수치뒤에 L로 표시
        float   / 4byte / 실수. 저장시 수치뒤에 f 로 표시
        double  / 8byte / 실수. 저장시 수치뒤에 d로 표시
        boolean / 1bit  / false, true 
        char    / 2byte / 한가지 '문자'. 문자열 아님 주의. ASCII코드 사용가능.
    Non-primitive [null값이 가능, 대문자시작, 모든종류가 같은 크기를 가짐]
        String  / 문자열 ""로 감싸서 저장한다
        Arrays  / 변수열
        Classes
        Interface
        ...

* System.out.println()
    화면에 출력시키는 매서드. () 안에 출력시키고 싶은 문자, 또는 변수를 지정한다.
    일반문자열은 ""로 구분하고, 숫자는 그대로 쓰고, 변수는 이름을 지정해주면 된다.
    +, -, /, * 등의 연산자를 사용할 수 있다.
        숫자 + 숫자 = 덧셈된 값
        숫자 + 문자 = 숫자와 문자를 나열해서 표시
        문자 + 문자 = 두 문자열을 나열해서 표시
        (이때 숫자나 문자는, 숫자나 문자를 담고있는 변수명으로도 대체가능)
    여러줄을 반복하여 사용할경우 자동으로 줄바꿈을 해준다.
* System.out.print()
    위와 동일한 매서드인데 다른점은 반복사용해도 줄바꿈을 해주지 않음.
