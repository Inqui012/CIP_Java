* CSS 속성값 선택자
    [속성] / 해당 속성을 가진 모든 태그
    [속성 = "값] / 해당 "값"을 가진 태그
    [속성 ~= "값"] / "값"을 포함하는(하이픈 비포함, 단어단위) 속성값을 가진 태그
    [속성 |= "값"] / "값"을 포함하는(하이픈 포함, 단어단위) 속성값을 가진 태그
    [속성 *= "값"] / "값"을 포함하는(글자단위) 속성값을 가진 태그
    [속성 ^= "값"] / "값"으로 시작하는 속성값을 가진 태그들
    [속성 $= "값"] / "값"으로 끝나는 속성값을 가진 태그들
